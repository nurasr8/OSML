# OSML Launcher

**OSML (Open Source Minecraft Launcher)** is a free and open-source Minecraft launcher written in **C++** with **gtkmm 4** for a native desktop experience on Linux and other platforms supported by GTK.

OSML aims to provide a lightweight, modern, and fully open launcher capable of managing and launching multiple Minecraft versions and modding platforms while respecting Mojang's authentication requirements.

---

## Features

* Modern desktop interface built with **pygobject**
* Written entirely in **python 3.10**
* Support for:

  * Vanilla Minecraft
  * Forge
  * NeoForge
  * OptiFine
* Version selection interface
* Profile and username management
* Launch log and status output
* Open Minecraft directory directly from the launcher
* Open-source

---

## Supported Versions

OSML supports a wide range of Minecraft releases, including:

* Vanilla versions
* Forge versions
* NeoForge versions
* OptiFine versions

---

## Authentication

OSML is a pirate Minecraft launcher so you don't need microsoft account.
